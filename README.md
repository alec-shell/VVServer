This is the back-end REST server for the VintageVinyl application, built on the Springboot framework.
- DEPLOYMENT: Runs on a private cloud VM.
- SECURITY: Https-enabled, operating behind a Cloudflare proxy.
- FUNCTIONALITY: Acts as a proxy server between the VintageVinyl client and the Discogs API, handling:
    - OAuth authentication
    - Client requests
    - API responses

* Link to VintageVinyl repo: https://github.com/alec-shell/VintageVinylTracker
